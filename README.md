# 🤖 Simple AI Chatbot (Python)

## 🚀 Project Overview

This project is a **rule-based chatbot** developed using Python.
It interacts with users by recognizing keywords in user input and responding with predefined answers using basic Natural Language Processing (NLP) techniques.

---

## 🧠 Features

* 💬 Interactive chatbot (CLI-based)
* 🔍 Keyword-based intent recognition
* 📁 JSON-based response storage
* ⚡ Fast and lightweight
* ⚠️ Exception handling for smooth execution

---

## 🏗️ Concepts Used

* Functions
* File Handling (JSON)
* Exception Handling
* List Comprehension
* Basic NLP (keyword matching)

---

## ⚙️ How It Works

1. User enters input
2. Input is converted to lowercase
3. Program matches keywords from JSON file
4. Returns predefined response

---

## ▶️ How to Run

1. Clone the repository:

```
git clone <your-repo-link>
```

2. Navigate to project folder:

```
cd chatbot
```

3. Run the program:

```
python chatbot.py
```
